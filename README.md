# IFSP Cafeteria Management Panel
A React-based administrative dashboard designed to manage orders, products, and operational settings for a school cafeteria. This project was developed as a collaborative effort by Pedro Eugelmi and Alisson Zaramello.

## 🍎 Project Purpose
This panel serves as the "back-office" for the cafeteria application, allowing administrators to:

Manage Inventory: Create and categorize products (snacks, drinks, etc.).

Order Tracking: Monitor and update the status of student/staff orders in real-time.

Operational Control: Set opening/closing hours and manage user permissions.

User Management: Oversee the accounts of both cafeteria staff and clients.

## 🛠️ Key Features
Authentication: Secure login flow integrated with Axios interceptors.

Dynamic Filtering: Advanced filtering for products and orders to improve efficiency during rush hours.

Opening Hours Logic: A dedicated module (createOpeningHours) to manage when the cafeteria is accepting orders.

Category System: Hierarchical organization of the menu for better user navigation.

## 🚀 Getting Started
Prerequisites
Node.js (LTS recommended)

npm or yarn

## Installation
Clone the repository

## Install dependencies:
npm install

npm start: Runs the app in development mode at http://localhost:3000.

npm run build: Bundles the app for production in the build folder.

npm test: Launches the test runner.
